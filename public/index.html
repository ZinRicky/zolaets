Welcome to Zolaets!

Zolaets is a rough recreation of the flagship Zonelets style as a Zola theme. The main objective of this theme is to make the transistion from Zonelets to Zola as seamless as possible (for the Web visitors). This theme does not aim for 100% fidelity to the original neither on the coding nor on the final result.

About the original

Zonelets is a semi-automatic blog builder. Its main gimmick is being completely manual except for a single JS script, which keeps track of the published posts.
